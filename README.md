# Photography Portfolio Website

A responsive photography portfolio website built with HTML, CSS, and JavaScript. This website is designed to showcase photography works in an elegant and user-friendly manner.

## Features

- Responsive design that works on all devices
- Image gallery with category filtering
- Lightbox for viewing images in full size
- Contact form for potential clients to reach out
- About section to share your story as a photographer
- Modern and clean UI/UX

## Structure

- `index.html` - Main HTML file
- `css/styles.css` - CSS styles for the website
- `js/script.js` - JavaScript for interactivity
- `images/` - Directory for storing photography images
- `generate-placeholders.html` - Helper file to generate placeholder images

## How to Use

1. Clone this repository
2. Replace the placeholder images with your actual photography works
3. Update the personal information in the About and Contact sections
4. Customize the colors and styles to match your brand
5. Deploy to your preferred hosting service

## Customization

### Adding Your Photos

1. Replace the placeholder URLs in `js/script.js` with paths to your actual images
2. Update the titles and descriptions for each image
3. Add or remove categories as needed

### Changing Colors

The website uses CSS variables for colors, which can be easily changed in the `:root` section of `css/styles.css`:

```css
:root {
    --primary-color: #333;
    --secondary-color: #f4f4f4;
    --accent-color: #4caf50;
    --text-color: #333;
    --light-text: #fff;
    --dark-bg: #222;
    --transition: all 0.3s ease;
}
```

### Updating Personal Information

Update the following sections in `index.html`:

- Website title and meta description
- Hero section text
- About section content
- Contact information
